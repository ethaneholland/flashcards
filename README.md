# Iterum
COMP 4020 Group 5: Flashcard Collage Prototype

A flashcard collage prototype built for COMP 4020 at the University of Manitoba. Iterum lets users arrange and connect idea cards on a freeform board, then iterate on them across multiple versioned canvases.

---

## Features

### Subjects and Homepage

The homepage displays subject cards, each subject contains its own board and revision history. 

  - **Limitation**: There is only one working subject board at this time.

### Collage Board

Add cards to a board with custom text, drag them around and connect related cards.

### Versioning

 - Click New in the header to create a new canvas version from the current one
 - Hidden cards are dropped
 - Locked cards are carried over to the new canvas version.
 - Cards marked for revision will be carried over and highlighted until edited.
 - Add new cards or rearrange existing cards on the new canvas
 - Each version is labelled iterò 1, iterò 2, etc. and each iteration of the canvas is saved.

## Running development version
Project was set up with [https://www.w3schools.com/react/react_getstarted.asp](https://www.w3schools.com/react/react_getstarted.asp)

#### Steps
 - Download Node.js
 - Clone repository
 - `cd` into flashcards-app
 - `npm install`
 - `npm run dev`
